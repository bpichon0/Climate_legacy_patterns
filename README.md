> Contact: Benoît Pichon, **benoit.pichon0@gmail.com**

This repository contains the code used to perform the analyses for both main text and supplementary information.
All the code was made on R (*v4.4.1*).
All packages needed are available in *Legacy_functions.R*. If you don't have all packages installed, please uncomment line 5 of the *Legacy_functions.R* script to install them.


All analyses and figures are made using the *Make_figs.R* script, which is divided into different chunks of code, each corresponding to a different analysis/figure.
